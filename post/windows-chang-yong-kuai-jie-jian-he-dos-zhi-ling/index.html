windows常用快捷键和dos常用指令

发布于 2021-09-25

windows常用快捷键

Ctrl+c 复制

Ctrl+v 拷贝

Ctrl+a 全选

Ctrl+z 撤销

Ctrl+x 剪切

Ctrl+s 保存

Alt+F4 关闭窗口

Windows+r 输入cmd回车打开命令窗口

Windows+e 打开我的电脑

Ctrl+Shifit+Esc打开任务管理器

Shift+Delete 永久删除

进入命令窗口的方法

1.windows+r输入cmd回车

2.资源管理器地址栏加上cmd路径

3.在任意文件夹下面按shift+鼠标右键点击，在此处打开命令窗口

基本Dos命令
1.切换盘符 d: (盘的位置加英文冒号）

2.清屏 cls

3.退出命令行窗口 exit

4.查看当前目录所有文件 dir

5.返回上级目录 cd..

6.进入指定文件夹 cd目录（cd加上你要进入的目录）

7.查看电脑IP ipconfig

8.打开应用 计算机 calc

画画本 mspaint

记事本 notepad

9.获得网址IP信息 ping加上网址

10.创建文件夹 在桌面的目录下输入md test

在文件夹里面创建文本cd >a.txt

删除 del a.txt

删除文件夹 rd test
